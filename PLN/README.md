# Projetos de Processamento de Linguagem Natural

Este repositório reúne os projetos desenvolvidos no âmbito da Unidade Curricular de Processamento de Linguagem Natural do Mestrado em Engenharia Biomédica (Universidade do Minho, 2024/2025).

O repositório está organizado em duas pastas principais, correspondentes às fases de evolução do projeto:
* TP1 - Extração e Estruturação: Foca-se na conversão de documentos médicos não estruturados (PDF) em dados organizados (JSON), utilizando expressões regulares e _pipelines_ de limpeza de texto.
* TP2 - Enriquecimento e Aplicação _Web_: Expande o _dataset_ anterior através de _Web Scraping_, processamento semântico com modelos de linguagem (BERT) e disponibiliza os dados numa plataforma Web interativa desenvolvida em _Flask_.
